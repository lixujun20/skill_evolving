Step 17:
Thought: I realized I should encapsulate the backtesting logic into a reusable function, as I might want to test different strategies (e.g., EMA instead of SMA) in the future. I'll define a function `backtest_strategy`.
Code:
```python
def backtest_strategy(df, initial_capital=10000):
    capital = initial_capital
    shares = 0
    trade_log = []
    
    for i in range(len(df)):
        signal = df.loc[i, 'Signal']
        price = df.loc[i, 'Close']
        date = df.loc[i, 'Date']
        
        if signal == 'Buy' and capital >= price:
            shares_to_buy = capital // price
            cost = shares_to_buy * price
            capital -= cost
            shares += shares_to_buy
            trade_log.append({'type': 'Buy', 'price': price, 'date': date, 'shares': shares_to_buy})
            
        elif signal == 'Sell' and shares > 0:
            revenue = shares * price
            capital += revenue
            trade_log.append({'type': 'Sell', 'price': price, 'date': date, 'shares': shares})
            shares = 0
            
    final_value = capital + (shares * df.iloc[-1]['Close'])
    return final_value, trade_log

# Test the function
final_val, logs = backtest_strategy(df)
print(f"Function Backtest Final Value: {final_val}")
```

def calculate_indicators(df):
    df['SMA_7'] = df['Close'].rolling(window=7).mean()
    df['SMA_30'] = df['Close'].rolling(window=30).mean()
    df['EMA_7'] = df['Close'].ewm(span=7, adjust=False).mean()
    df['EMA_30'] = df['Close'].ewm(span=30, adjust=False).mean()
